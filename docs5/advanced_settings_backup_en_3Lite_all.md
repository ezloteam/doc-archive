Backup

If you check the "Backup my configuration at cp.mios.com" option, Vera will
automatically backup your configuration online each night so that if Vera
breaks and we send you another one, you can go to cp.mios.com, download the
last backup file, and restore it so you don't need to setup Vera again.

To manually backup Vera, click 'Create Backup'. Vera will give you a backup
file to save to your PC.

To restore a backup file, either one that you created manually with 'Create
Backup', or an automatic backup at cp.mios.com, click the backup file on the
right and download it on your computer, select it, and click 'Restore'.  

If you select Restore Dongle Firmware, all the data stored on the Z-Wave chip
will be restored including the Z-Wave version, role, home id, devices, etc.

You can also click 'Restore to Factory Default' to reset Vera's configuration
just like it was when Vera first arrived.
